# Agent Materials

## Purpose

Prompts, instructions, and other agent-oriented working materials used by the
team while developing FathomDB.

## Keep here

- reusable agent prompts
- structured review or harvesting tasks
- internal workflow aids for agent-assisted work

## Do not keep here

- public docs
- accepted product requirements
- disposable scratchpad notes better suited for `dev/tmp/`

## Canonicality

Canonical for agent workflow materials only.

## Lifecycle

Update as workflows evolve; archive or delete obsolete prompts deliberately.
