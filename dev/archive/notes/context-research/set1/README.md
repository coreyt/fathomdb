# Tmp Scratchpad

## Purpose

Disposable scratch space for drafts, experiments, temporary comparisons, and
ad hoc notes.

## Keep here

- rough notes
- throwaway diagrams
- migration sketches
- temporary evaluation artifacts

## Do not keep here

- accepted ADRs
- canonical requirements
- final architecture docs
- public documentation
- release-critical checklists

## Canonicality

Non-authoritative.

## Lifecycle

Contents may be deleted, moved, or rewritten at any time without notice.
