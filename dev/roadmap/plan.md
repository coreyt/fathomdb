# Post-0.6.0 Roadmap

`dev/roadmap/` is for **0.7.0 and later** only.

0.6.0 implementation planning does **not** live here. The canonical 0.6.0
plan is:

- `dev/plans/0.6.0-implementation.md`

Current future-release roadmap files:

- `dev/roadmap/0.8.0.md` — explicitly deferred security and posture work that
  is not part of 0.7.0

Rules for this directory:

- keep only 0.7.0+ items here
- do not park active 0.6.0 packet plans here
- if a 0.6.0 item is still being implemented, tracked, or deferred within the
  0.6.0 release line, keep it under `dev/plans/` or `dev/progress/`
- if a 0.6.0 item is explicitly postponed to a future release, move it here
  under the target release file
