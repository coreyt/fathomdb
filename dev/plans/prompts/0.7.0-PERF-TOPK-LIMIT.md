# Slice 0.7.0-PERF-TOPK-LIMIT — Top-K LIMIT caller-spec change

Caller-spec change on the search verb. Per HITL Q4 (additive),
default behavior is unchanged from 0.6.x (unlimited result set);
caller may opt in to `LIMIT k` to cap result-set traversal. This
slice lands the surface in Rust + Python + TypeScript + CLI and
gates a tighter AC-012 sub-budget on the LIMIT-on shape.
