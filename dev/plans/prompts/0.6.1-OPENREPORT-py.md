**Shape decision (locked HITL 2026-05-24, supersedes earlier wrapper
plan):** OpenReport is exposed as an **engine-attached accessor
method** `engine.open_report()`, NOT as a return-shape change on
`Engine.open`. `Engine.open` keeps its 0.6.0 signature
(`Engine.open(...) -> Engine`); the report is read after open via
`engine.open_report() -> OpenReport`. Sibling semantics to
`engine.counters()` (runtime-state snapshot) — `open_report()` is
the open-time-state snapshot. Pros: zero risk to legacy callers,
clean type-checker story, slots into the existing engine-attached
instrumentation taxonomy. Per `dev/plans/0.6.1-implementation.md`
§ "Q-002 resolution (2026-05-24)" for full rationale.

## Hard constraints

- **TDD red-green-refactor** per `feedback_tdd`. Failing test first.
- **No native (`fathomdb-engine`) code changes.** Native shape is
  already correct (`lib.rs:541-548`). Bindings surface what's
  already there; do not extend or rename native fields.
- **No change to `Engine.open` signature.** `Engine.open(path,
**config)` continues to return an `Engine`. Existing callers
  `engine = Engine.open(path)` work unchanged. This is the
  Shape-D guarantee — do not regress to a wrapper-class return.
- **Native field names are authoritative.** Surface them on the
  Python `OpenReport` under the exact snake_case identifiers
  (`schema_version_before`, `schema_version_after`,
  `migration_steps`, `embedder_warmup_ms`, `query_backend`,
  `default_embedder`). Do not invent paraphrases; do not omit the
  extended native fields.
- **Idempotent accessor.** `engine.open_report()` must return the
  same data on every call — the report is captured once at open
  time. Do not re-compute, re-probe, or live-refresh.
- **No new ADR.** OpenReport semantic + the engine-attached-method
  taxonomy are already locked in `dev/design/engine.md` +
  `dev/interfaces/python.md`. Shape D's "accessor not return"
  decision is an interface-doc edit, not an ADR-class change.
- **Symmetry obligation.** ts slice mirrors this shape
  (`engine.openReport(): Promise<OpenReport>` per
  `dev/interfaces/typescript.md` camelCase + Promise conventions).
