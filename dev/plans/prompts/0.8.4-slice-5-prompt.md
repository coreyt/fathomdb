## Deliverables (TDD, RED → GREEN; pure-Python, no LLM, no native-extension rebuild)

1. **Corpus loader** — `src/python/eval/apnews_corpus.py`:
   - Read `raw_data.zip` from the manifest dir → an in-memory list of articles
     `{doc_id, body, ...}`. Verify `len == manifest["n_articles"]` (1397) and the zip
     **sha256 matches** `manifest["raw_data_zip_sha256"]` (a corpus-validity guard — fail
     loudly on mismatch). Expose a constant/env for the corpus root; default to the path
     above but allow override (so the test can point at a tiny fixture).
2. **AutoQ loader** — in the same module or `eval/apnews_autoq.py`:
   - Load the bundled question sets into a typed question list tagged by **bucket**
     (`activity_global`, `activity_local`, `data_global`, `data_local`, `data_linked`),
     preserving v2 assertions where present (for the later length-bias corroboration).
   - A **coverage** accessor: questions span local↔global with **no empty bucket**.
3. **Baselines** (both conform to `RetrievalAdapter` over the AP-News corpus):
   - `eval/baselines_084.py::VectorRagAdapter` — a standalone **dense** retriever
     (reuse whatever eval-side embedding utility already exists; if none, a small
     deterministic embedder is fine — this is the BASELINE, keep it independent of the
     FathomDB engine to avoid conflating baseline with SUT). `retrieve(q,k)` → top-k Hits.
   - `eval/baselines_084.py::LongContextAdapter` — the "stuff-it-all-in" control:
     returns the corpus packed to a **window-fit budget** (a char/token budget param;
     deterministic ordering) as Hits, so the shared answerer can stuff the window. No
     ranking model — it is the honest upper-bar control.
4. **Tests** — `src/python/tests/test_apnews_corpus.py` + `test_baselines_084.py`:
   - corpus-validity: count + sha256 guard fire on a mismatch fixture (RED first).
   - AutoQ coverage: every bucket non-empty; data_linked present.
   - adapter conformance: both satisfy `isinstance(x, RetrievalAdapter)` (runtime_checkable),
     return `list[Hit]`, respect `k`, deterministic on a tiny fixture corpus.
   - Use **tiny fixtures** (a handful of synthetic articles/questions), NOT the real
     1397-article payload — tests must run with no network and no real corpus present.
