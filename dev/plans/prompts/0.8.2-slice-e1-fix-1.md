## 2. Mandate (exactly this)

   **before** `try_get_loaded()`. (Keep the existing `rerank_depth==0` early return in `rerank_fused`.)
   No model load on an empty result set.
2. **Cfg-gate the default-build test:** add `#[cfg(not(feature = "default-reranker"))]` to

   prefer, ALSO add a feature-on test asserting "weights-absent ⇒ soft-fallback even feature-on" — optional,
