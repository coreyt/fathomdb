### Campaign context (already on `origin/main`)

The default embedder is now real and shipping. From `dev/design/embedder-decision.md`:

- **Model**: `BAAI/bge-small-en-v1.5` (HF snapshot `5c38ec7c...`); dim 384.
- **Runtime**: candle-transformers `bert::BertModel`, in-process.
- **Pipeline**: WordPiece tokenization (max 512 tokens) → mean-pool over
  attention mask → L2-norm → mean-centering (corpus-mean f32 vector
  subtracted before sign-quantization) → `vec_quantize_binary` sign-bit
  → bit-KNN top-K=192 → f32 `vec_distance_l2` rerank → top-10.
- **Bindings**: Python `use_default_embedder=True` and TypeScript
  `useDefaultEmbedder: true` opt into the default. `OpenReport` exposes
  `embedder_events` (typed-union per EU-6 FIX-2), `embedder_warmup_ms`,
  `embedder_download_ms`.
- **K = 192** is **locked** (`dev/design/embedder-decision.md` § 4.3).
  Do not change K inside this slice; the locked value is the production
  default and the measurement must reflect production behavior.

### Suggested measurement plan

1. **Seeder**. Add an alternate `seed_ac013_corpus_real(engine: &Engine,
   n: usize) -> Duration` in `perf_gates.rs` (or a helper module) that
   reads up to `n` docs from `data/corpus-data/raw/*.jsonl` via the
   existing `corpus_subset` plumbing and batch-writes them through

   batched writes now produce one vec0 row per doc). Use the real
   default embedder by opening the engine with
   `EmbedderChoice::Default`. Cache embeddings on disk to avoid
   re-embed cost across runs.

2. **Query set**. Reuse the EU-0 methodology: synthesize 100 queries
   from the same corpus subset (title-or-lead-sentence, with
   `target_doc_idx` exclusion to prevent self-retrieval). Save the
   query set + seeds deterministically so PR-3 can re-run on the same
   set at canonical-CI.

3. **N targets**. At minimum: N = 1000 and N = 10_000 dev-box runs.
   If wall-time + cache allow, N = 100_000 is a useful intermediate
   between dev-box and PR-3's canonical N=1M. Skip N=1M in EU-7;
   that's PR-3.

4. **Measurements per N**:
   - AC-013 p50 / p99 query latency (1000 measurement samples, after
     warmup) against the locked 80/300 ms budget. **Expected to pass**

     Hamming + K=192 rerank are sub-ms.
   - AC-013b recall@10 with bootstrap 95% CI (1000 resamples).
     **Anchor number for PR-2**. Compare against current `0.90` floor
     for the honesty report.

   recall_at_10=… recall_ci_lo=… recall_ci_hi=…` line per N so STATUS
