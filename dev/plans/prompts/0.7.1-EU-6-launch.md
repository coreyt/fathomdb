### GREEN commit — implementation

#### Python (PyO3)

Wire `use_default_embedder=True` to `EmbedderChoice::Default` Rust-side.
Wire `False` (the default) to `EmbedderChoice::None` (engine opens;
vector writes fail with `EmbedderNotConfigured` — same as today's
null-embedder behavior).

```python
# Caller surface (Python)
engine = Engine.open(path, use_default_embedder=True)
```

The PyO3 side translates that to:

```rust
let choice = if use_default_embedder {
    EmbedderChoice::Default
} else {
    EmbedderChoice::None
};
Engine::open_with_choice(path, choice)
```

Surface the four new `OpenReport` fields via the existing `OpenReport`
PyO3 class:

```python
# Caller surface
report = engine.open_report
report.embedder_download_ms   # Optional[int]
report.embedder_events        # List[EmbedderEvent]
report.embedder_mean_centering_required  # bool
report.embedder_mean_vec_pinned          # bool
```

If `EmbedderEvent` isn't yet exposed in PyO3, expose it as a `PyAny`-
typed list with each event a dataclass-style dict (`{"kind":
"DefaultEmbedderDownload", "url": "...", "bytes": ..., "sha256": "...",
"cache_path": "..."}`). The exact shape is a judgment call; pick the
most ergonomic Python surface that doesn't lose information.

#### TypeScript (NAPI)

Add `useDefaultEmbedder?: boolean` to `OpenOptions`. Default `false`.

```typescript
// Caller surface
const engine = await engineOpen(path, { useDefaultEmbedder: true });
```

Same Rust translation as Python.

Surface the four new `OpenReport` fields via the existing TS `OpenReport`
shape (camelCase per NAPI convention):

```typescript
report.embedderDownloadMs   // number | null
report.embedderEvents       // EmbedderEvent[]
report.embedderMeanCenteringRequired  // boolean
report.embedderMeanVecPinned          // boolean
```

## Default embedder

`Engine.open(path, use_default_embedder=True)` opts into the engine's
default embedder (`fathomdb-bge-small-en-v1.5`). On first use, weights
are downloaded from HuggingFace and cached under
`~/.cache/fathomdb/embedders/`; subsequent opens hit the warm cache.

`OpenReport.embedder_events`).

`use_default_embedder=True` (engine's bge-small) or `False` (no
embedder; vector writes fail with `EmbedderNotConfigured`).
