## A. Reframed E0a — frontier instrumentation as the product (BLOCK-1)
Root cause (confirmed in code): the graph arm seeds from the top-`SEED_N`=10 two-arm
hits = **doc nodes**, written with `logical_id=None` (`lib.rs:2635`); the seed
resolver (`lib.rs:5379-5397`) gets `Some(None)` → the `Some(Some(lid))` guard fails →
frontier empty → zero graph candidates. Even with a doc `logical_id`, edges connect
*entity* logical_ids; no doc→entity edge exists. So `source_id` is **necessary but
inert**. Deliverable = the measurement that proves this, plus the carrier.

Add a side-channel struct (NOT a new `SearchResult`/`SearchHit` field — byte stability):
```
pub struct GraphFrontierStats {
    pub seeds_considered: u32,       // hits inspected in take(SEED_N), skipping TextEdge
    pub seeds_resolved:   u32,       // matched Some(Some(lid)) at :5392
    pub frontier_nonempty: bool,
    pub graph_candidates_emitted: u32,
}
```
`resolved_seed_rate = seeds_resolved / seeds_considered` (define `0/0 → 0.0`).
`bfs_graph_arm_candidates` (`:5357`) returns `(Vec<SearchHit>, GraphFrontierStats)`;
thread stats through `search_inner`'s internal outcome, keep OFF `SearchResult`.
Expose via a `_graph_frontier_stats_for_test(query)` seam (mirrors the sanctioned
`_configure_vector_kind_for_test` PRE-1 pattern) — keeps it off the governed surface.

## B. Traversed-edge `source_id` (BLOCK-2)
Discard the memo's arbitrary `LIMIT 1` incident-edge lookup (non-deterministic,
under-credits multi-session entities). Carry the **traversed edge's `source_id`**:
1. `SearchHit` (`lib.rs:1041`) gains `pub source_id: Option<String>`.
2. Add `e.source_id` to the BFS `edge_stmt` SELECT (`:5405-5413`, already runs per node).
3. Neighbor map (`:5432-5438`) carries `(neighbor_lid, Option<String> edge_source_id)`.
4. Emit site (`:5459`) sets `source_id` from the traversing edge; `None` fallback kept.
5. All four two-arm construction sites set `source_id: None` (compiler-forced — no `..`).
Eval wiring: `r2_parity_eval.doc_id_of` prefers `sh.source_id` (canonicalized, §E)
when present, else the cursor map.
