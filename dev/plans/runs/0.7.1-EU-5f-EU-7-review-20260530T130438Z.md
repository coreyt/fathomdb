## What passed on inspection

- **Concurrency fix is coherent.** `commit_gate` serializes the full projection
  commit body; the threshold-crossing row does `add -> count -> take` under the
  accumulator lock; the pin tx updates `mean_vec` and re-quantizes all committed
  rows in the same tx; later rows center on insert. No interleaving leaves a row
  permanently uncentered, and no double-pin. Refs: `lib.rs:174, 3533, 3565,
  3620, 3682`.
- **Locking is ordered, not cyclic.** New path takes `commit_gate ->
  mean_accumulator -> SQLite tx`; no inverse path waits on `commit_gate` while
  holding the accumulator/state locks; state cleanup stays outside the gate.
  Refs: `lib.rs:174, 3079, 3533`.
- **`catch_unwind` addresses the scheduler wedge.** On panic the batch is marked
  terminal `ProjectionPanic`, `active_jobs`/`in_flight` cleanup runs
  unconditionally, no infinite re-enqueue. Worker connection reuse is safe
  (embedder panics precede SQL; commit-path panics unwind the local tx). Refs:
  `lib.rs:3120, 3134, 3177, 3656`.
- **Recovery pin is idempotent and runs before workers exist.** `open_locked`
  repairs only `MC-required && mean_vec NULL && rows >= threshold`, sets the
  open-report pinned flag, and only then does `ProjectionRuntime::new` decide
  whether to allocate a live accumulator. Refs: `lib.rs:1473, 1511, 488, 519,
  3694`.
- **Truncation matches the locked BGE contract.** `with_truncation(max=512)`
  before `encode(input, true)` keeps specials tokenizer-managed (not sliced);
  the added >512-token test asserts success at dim 384. Refs: `candle_bge.rs:50,
  114, 160`, `candle_bge tests:130`, `embedder-decision.md:117`.
- **No regression to retrieval/write contracts.** Search still uses
  `TOP_K_BIT_CANDIDATES = 192`; cursor advances after terminal-state recording;
  the `write_vector_for_test` pin block is unchanged. Refs: `lib.rs:2140, 2744,
  2932, 3682`.
- **EU-7 reporting is honest.** Harness does not re-pin the 0.90 floor or K, uses
  same-model f32 GT, excludes the target from both GT and production, mirrors AC
  budgets; output JSON plainly reports `0.828 < 0.90` with `R - 2*sigma = 0.795
  -> 0.79`. No floor gerrymandering or quiet budget relaxation. Refs:
  `eu7_real_corpus_ac.rs:24, 35, 123, 399, 701, 731`, `0.7.1-EU-7-output.json:31,
  37, 45, 55`.
- **Shipped choices match `embedder-decision.md`**: bge-small, dim 384,
  truncation 512, no silent recompute, K=192. The open-time repair is a
  crash-recovery exception, not a general recompute-on-open policy. Refs:
  `embedder-decision.md:55, 117, 232, 297`.
