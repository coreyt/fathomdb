# Dependency Evaluations

## Purpose

Scope and ownership notes for third-party dependency review and dependency
policy.

## Keep here

- dependency adoption criteria
- audit summaries
- links to per-dependency evaluation files
- notes about what is currently in scope for review

## Do not keep here

- lockfiles or package manifests
- generated audit output dumps with no summary
- public compatibility docs

## Canonicality

This README is canonical for directory scope. The detailed current audit index
is `index.md`.

## Lifecycle

Update when dependency policy or audit process changes.
