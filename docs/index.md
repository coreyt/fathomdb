# FathomDB Public Docs

FathomDB is a local-first retrieval and graph-oriented data system designed for
application and agent workloads. This site is the public documentation source
for users, operators, and SDK consumers.

> **Status: 0.6.0 pre-release.** Engine + bindings + release-engineering
> substrate complete (Phase 11 closed 2026-05-17). Durability harnesses,
> security fixtures, post-publish smoke, and external-user docs land in
> Phase 12 before the `v0.6.0` GA tag. See `release-notes/0.6.0.md` for a
> preview and the deferred-items list.

Use these sections:

- `getting-started/` for initial setup and orientation (preview install paths)
- `guides/` for task-based workflows (post-GA)
- `concepts/` for durable public model explanations (post-GA)
- `reference/` for surface-level API and packaging notes (post-GA)
- `operations/` for runtime and maintenance guidance (post-GA)
- `positions/` for consumer-relevant technical positions
- `release-notes/0.6.0.md` for the 0.6.0 preview + deferral disclosures
- `compatibility/` for cross-version compatibility notes
